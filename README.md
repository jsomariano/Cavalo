> 🐴 Cavalo! 🐴

Play famous meme "Cavalo" sound.

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g cavalo

## Running
Run command:

    npx cavalo

Or import in NodeJS script:

    import cavalo from 'cavalo';

    cavalo();
